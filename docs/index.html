What Makes It Different

🏠

No Cloud, Ever

The Gemma 4 2.3B model runs entirely on your device using LiteRT. Your conversations never touch a server.

💬

Plain Language Control

Tell your phone what to do in natural language. PokeClaw translates intent into taps, swipes, and text entry.

🔗

Works With Any App

Built on Android Accessibility Services. If you can use it, PokeClaw can control it.

🎯

No Account Needed

Download, install, run. No signup, no API key, no subscription. Yours permanently.

📖

Open Source

Apache 2.0 licensed. Read the code, fork it, build on it. No black box.

🤖

Gemma 4 Model

Powered by Google's Gemma 4 2.3B. Small enough for a phone, capable enough to be useful.
