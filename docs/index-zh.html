常見問題

支援咩手機？

Android 9+ arm64 架構。2B 模型需要大約 4 GB 可用 RAM。平價機都可以跑，但 CPU 模式 warmup 大概要 45 秒。有 ML 晶片嘅旗艦機（Pixel 8+、Galaxy S24、Dimensity 9200+）用 GPU 加速快好多。

真係免費？有咩隱藏條件？

冇隱藏條件。Apache 2.0 授權，完全 open source。冇廣告、冇收集數據、冇 premium 層級、冇訂閱費。完整代碼喺 GitHub 上。你可以自己 build source。

同 Google Assistant 有咩分別？

Google Assistant 只支援有限嘅 Google 整合 app，而且需要上網。PokeClaw 透過 Accessibility Services 控制你手機上任何 app，完全離線都可以用。佢讀取螢幕上真實嘅 UI 元素，自己搵出要點 tap。

真係可以離線用？

係。一次性下載 2.6 GB 模型之後，全部喺你部手機跑。你可以完全封鎖 PokeClaw 的網絡存取，照樣正常運作。本地模式零網絡請求。

我嘅數據安全嗎？

模型完全喺你嘅裝置上跑。你嘅對話、訊息同螢幕內容永遠唔會離開你嘅手機。app 本身冇伺服器、冇 telemetry、冇 analytics。代碼係 open source，你可以自己核實。

可以用雲端模型嗎？

可以。PokeClaw 支援任何 OpenAI 相容 API（GPT-4o、Claude、Gemini 等）。喺設定入面填入你自己嘅 API key 就得。雲端模式對複雜任務更智能。本地同雲端可以喺對話途中隨時切換。

Samsung 話係惡意軟件？

係 sideload 嘅誤報。Samsung 會標記任何喺 Play Store 以外安裝嘅未簽名 APK。呢個 app 完全 open source，本地模式零數據收集、零網絡請求。你可以自己 build source 核實。

準備好喺你部手機試 AI 未？

download APK，開一個權限，用普通語言控制你部手機。唔需要登記、唔需要 API key、唔使交月費。
